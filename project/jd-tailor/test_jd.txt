【职位】AI 产品经理
【公司】深圳某 AI 科技公司（B 轮）
【地点】深圳·南山
【薪资】15-25k · 13薪

## 岗位职责

1. 负责 AI Agent 产品线的规划与落地，撰写 PRD、用例、验收标准；
2. 深度参与 Prompt Engineering 方案设计，协同算法团队优化模型调用链路；
3. 主导用户调研，洞察 ToB 客户业务痛点，将需求转化为可执行的产品方案；
4. 推动 AI 工具在内部业务流程中的落地与迭代，建立 AI 效果评估体系；
5. 跨职能协作（研发/商务/运营），协调资源保障产品交付节奏。

## 任职要求

1. 本科及以上学历，1-3 年产品经理相关经验；
2. 熟悉 LLM / Agent 技术原理，有 AI 产品从 0 到 1 的落地经验优先；
3. 具备清晰的结构化思维，能独立完成需求分析与产品文档；
4. 有 ToB 业务或企业级产品经验优先；
5. 英语读写流利优先。

## 加分项

- 有 AI Coding 工具使用经验（Cursor / Claude Code 等）；
- 参与过黑客松或 AI 产品竞赛；
- 熟悉飞书 / 钉钉等企业协同工具的产品逻辑。
