【职位】AI 产品经理（ToB 方向）
【公司】深圳某企业服务 SaaS 公司（C 轮，200-500 人）
【地点】深圳·福田
【薪资】18-28k · 14薪

## 岗位职责

1. 负责企业 AI 助手产品线（智能客服 + 知识库问答）的需求规划与版本迭代；
2. 深入 ToB 客户业务场景，访谈关键用户，将模糊业务诉求转化为可落地的产品方案；
3. 与算法团队共同设计 Prompt 工程方案，推动 LLM 在客服、审核等场景的稳定落地；
4. 建立 AI 效果评估体系（准确率、召回率、用户满意度），持续驱动产品指标提升；
5. 跨职能协作（研发/商务/实施），保障客户交付节奏与产品质量。

## 任职要求

1. 本科及以上，2-4 年产品经理经验，有 ToB 产品或 AI 产品落地经验优先；
2. 熟悉 LLM / RAG / Agent 技术原理，能与算法同学对齐产品边界；
3. 具备完整的需求分析与 PRD 撰写能力，有良好的结构化思维；
4. 有企业级客户访谈、需求挖掘经验；
5. 英语读写流利优先（部分客户为外资企业）。

## 加分项

- 有 AI Coding 工具使用经验（Cursor / Claude Code 等）；
- 参与过内部 AI 工作流建设或 Prompt Engineering 实践；
- 有招聘、HR Tech 或 CRM 类产品背景。
