**架构意义**：分身从"纯 Guide"升级为"Guide + Sensor"双机制，对应 Harness Engineering 框架中 feedforward + feedback 的完整闭环。

**启动规则（重要）**：`.claude/settings.json` 的权限配置只在以 `木安-AI分身/` 为根目录启动 Claude Code 时生效。必须用以下方式启动，否则 Edit/Write 每次都会弹确认框：
```bash

claude
```
或在 VSCode 中以该目录为工作区打开。**不要从其他目录启动后跨目录操作分身文件。**
