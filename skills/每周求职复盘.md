# Skill：每周求职复盘（定时触发）

> 每周日 21:00 自动触发，生成本周求职进展结构化复盘，写回 `memory/当前任务版.md`。

## 触发条件
- **定时**：每周日 21:00（Cron 自动触发）
- **手动**：木安说"帮我做本周求职复盘"

## 执行步骤

### 1. 读取当前状态
- 读 `memory/当前任务版.md`，提取本周投递、面试、进展记录

### 2. WebSearch 补充（可选）
- 如有新公司名出现，用 WebSearch 快速核实基本信息

### 3. 生成复盘报告

输出格式：
```
## 求职周报：[本周日期]

### 本周动作
| 公司 | 岗位 | 进展 | 下一步 |
|------|------|------|--------|
| | | | |

### 有效动作（值得继续做）
-

### 低效动作（下周减少）
-

### 本周卡点
-

### 下周重点（最多 3 项）
1.
2.
3.
```

### 4. 写回记忆
将复盘摘要更新到 `memory/当前任务版.md` 的"上次对话摘要"字段。

## 校准规则
- 没有投递/面试记录时，直接说"本周无新进展"，不凑内容。
- 不鼓励，不安慰，只输出结构化事实与行动项。
