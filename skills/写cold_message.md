# Skill：写 Cold Message

> 用于生成针对性的求职冷联系消息（LinkedIn / 邮件 / 微信）。

## 触发条件
木安提到"帮我写个消息给 HR"、"怎么联系这家公司"、或给出一个 JD 要求写冷联系。

## 核心原则
1. **真实 > 套路**——不用模板腔，不用"贵司""拜读"，用正常人说话的方式
2. **一条消息一个目的**——要么约聊、要么投递、要么追问进度，不混着来
3. **对方视角**——HR 每天收几十条消息，前两句决定生死

## 执行流程

### 1. 收集信息
需要知道：
- 目标公司 + 岗位
- 联系人角色（HR / 业务负责人 / 猎头）
- 联系渠道（LinkedIn / 微信 / 邮件）
- 木安跟这个岗位的核心匹配点（最多 2 个）

### 2. 生成结构
```
[一句话钩子：我为什么联系你 + 我跟这个岗位的关联]
[1-2 句核心匹配：具体经历/能力，不泛泛]
[一句话行动请求：明确你想要什么]
```

### 3. 渠道适配
- **微信/即刻**：控制在 3-5 句，口语化，不加"您好"开头
- **LinkedIn**：控制在 100 字内英文或 150 字中文，稍正式
- **邮件**：可以稍长，但正文不超过 200 字，主题行要有信息量

### 4. 输出格式
```
## Cold Message：[公司] [岗位]
渠道：微信 / LinkedIn / 邮件
版本：A（直接版）/ B（委婉版）

[消息正文]

### 发送建议
- 最佳发送时间：
- 如果没回复，X 天后跟进：[跟进话术]
```

## 校准规则
- 不写木安解释不了的内容。消息里提到的每个经历，面试时都要能展开。
- 不夸大。"有经验"和"精通"是两回事。
- 如果匹配度低于 60%，先说"这个岗位不建议冷联系"，再给理由。
